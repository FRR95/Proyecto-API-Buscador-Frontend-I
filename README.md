## Download ⬇

#### Open your terminal and:

**1-Clone this repository**

```sh
git clone https://github.com/FRR95/Proyecto-API-Buscador-Frontend-I.git
```

**2-Install the dependencies**

```sh
npm i
```

**3-Run the localhost website**


```sh
npm run dev
```
